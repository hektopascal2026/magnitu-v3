Magnitu — Install on a new Mac

1. Install Apple command-line tools

Open Terminal and run:

xcode-select --install

Finish the installer if a dialog appears. If it says the tools are already installed, continue.

2. Clone the repository

Use this path so the Magnitu.app shortcuts work (Desktop and Applications):

mkdir -p ~/Applications
git clone https://github.com/hektopascal2026/magnitu-v3.git ~/Applications/magnitu3
cd ~/Applications/magnitu3

You may clone elsewhere if you only use the browser; automatic Magnitu.app aliases are intended for ~/Applications/magnitu3.

3. Run the installer

bash install/bootstrap.sh

Follow the prompts: Seismo mothership URL, Magnitu API key, optional database reset, and model setup. At step 6 you can type back at prompts to return to the menu.

4. Start Magnitu

Magnitu.app: After bootstrap, double-click Magnitu on the Desktop or in Applications. If macOS blocks it: right-click → Open → Open once.

Terminal + browser:

cd ~/Applications/magnitu3
./start.sh

Open http://127.0.0.1:8000.

Desktop window (WebView):

./start_desktop.sh

5. If Magnitu.app is missing

cd ~/Applications/magnitu3
bash install/post_bootstrap_mac_app.sh

6. Updates

cd ~/Applications/magnitu3
git pull

Prerequisites: Git and Python (from Command Line Tools), a reachable Seismo index.php URL, and a Magnitu API key from Seismo. Optional: copy .env.example to .env and set GEMINI_API_KEY for Gemini labeling.

Logs: See install/README.txt if launch fails (e.g. ~/Applications/magnitu3/.magnitu_desktop_last.log).
