Model Architecture

Transformer uses a pre-trained language model for better understanding. TF-IDF uses keyword counting (faster, simpler).
HuggingFace model name. Default: xlm-roberta-base (multilingual, supports German/French). Change only if you know what you're doing.
Use CUDA on Linux/NVIDIA or Metal (MPS) on Apple Silicon for faster embedding computation.
Changing architecture requires retraining.
